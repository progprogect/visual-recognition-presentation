# Automated Nut Classification System - Presentation

Интерактивная веб-презентация системы автоматической классификации орехов с использованием компьютерного зрения.

## Структура презентации

Презентация состоит из следующих секций:

1. **Title Slide** - Главный экран с основными метриками
2. **Problem** - Описание проблемы заказчика
3. **Solution** - Обзор решения
4. **How It Works** - Демо-видео работы системы
5. **Real-World Examples** - Примеры обучения модели и результатов
6. **Implementation Options** - Два варианта реализации:
   - Вариант 1: Вибростол (ручная загрузка)
   - Вариант 2: Конвейерная лента (полная автоматизация)
7. **Analytics Dashboard** - Три варианта дашборда:
   - Overview - Общий обзор производства
   - Batch Details - Детали партии с доказательствами
   - Supplier Reports - Отчеты по поставщикам
8. **Technical Architecture** - Техническая архитектура системы
9. **Development Timeline** - Временные рамки разработки
10. **Business Value & ROI** - Бизнес-ценность и возврат инвестиций
11. **Future Expansion** - Возможности расширения (фотосепарация)
12. **Call to Action** - Призыв к действию

## Как открыть презентацию

1. Откройте файл `index.html` в современном веб-браузере (Chrome, Firefox, Safari, Edge)
2. Используйте навигационное меню вверху для быстрого перехода к секциям
3. Прокручивайте страницу вниз для просмотра всех секций
4. Используйте вкладки для переключения между вариантами реализации и дашбордами

## Особенности

- **Адаптивный дизайн** - работает на десктопе, планшете и мобильных устройствах
- **Интерактивные элементы** - переключение между вариантами, вкладки дашбордов
- **Визуальные диаграммы** - графики и диаграммы для наглядности
- **Плавная прокрутка** - smooth scroll между секциями
- **Современный дизайн** - градиенты, тени, анимации

## Технические требования

- Современный веб-браузер с поддержкой HTML5, CSS3 и JavaScript
- Локальный веб-сервер (опционально, но рекомендуется для корректной работы видео)

## Запуск через локальный сервер (рекомендуется)

Для корректной работы видео рекомендуется использовать локальный веб-сервер:

### Python
```bash
python -m http.server 8000
```

### Node.js (http-server)
```bash
npx http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`
